# Tanvi Kinger — Personal Portfolio Website

A fully responsive personal portfolio website built with pure HTML, CSS, and JavaScript — no frameworks. Designed with a mobile-first approach and a clean, minimal, professional aesthetic.

🔗 **Live Site:** [tanvi06-web.github.io/DecodeLabs-Internship.project1](https://tanvi06-web.github.io/DecodeLabs-Internship.project1/)

## About

This portfolio introduces me as a third-year Computer Engineering student focused on Java, Data Structures & Algorithms, and web development fundamentals. It highlights my skills, projects, and provides a way to get in touch.

## Features

- Fully responsive layout (mobile, tablet, desktop)
- Clean, minimal, professional design with a soft pastel color scheme
- Smooth scrolling navigation
- Scroll-triggered section animations
- Mobile hamburger menu
- Hover and click interaction effects
- Contact form with client-side validation
- Back-to-top button

## Sections

1. **Hero** — Introduction and call-to-action
2. **About** — Background and current focus
3. **Skills** — Core technical skills
4. **Projects** — Featured projects with GitHub links
5. **Contact** — Contact form and social links

## Tech Stack

- **HTML5** — Semantic markup (`header`, `nav`, `main`, `section`, `footer`)
- **CSS3** — Grid (page layout) + Flexbox (components), mobile-first media queries
- **JavaScript (Vanilla)** — DOM interactivity, IntersectionObserver for scroll animations

## Project Structure

```
Project-1-Portfolio/
├── index.html      # Page structure and content
├── style.css       # Styling, layout, responsive design
└── script.js       # Interactivity (nav, animations, form handling)
```

## Run Locally

Just open `index.html` in any browser — no build step or server required.

For live-reload while editing, use the **Live Server** extension in VS Code:
right-click `index.html` → **Open with Live Server**.
